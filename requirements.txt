# ===== 安装说明 =====
# 1. 基本安装：pip install -r requirements.txt
# 2. GPU支持：需要安装对应CUDA版本的PyTorch
#    - CUDA 11.8: pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
#    - CUDA 12.1: pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
# 3. 在某些Linux系统上可能需要额外安装：
#    - sudo apt-get install python3-tk python3-dev
# 4. 如果使用conda环境：
#    - conda install tk opencv numpy matplotlib
# 5. Tesseract OCR需要单独安装系统级组件：
#    - Windows: 已配置路径 D:\Python\tool\Tesseract-OCR
#    - Linux: sudo apt-get install tesseract-ocr tesseract-ocr-chi-sim
#    - macOS: brew install tesseract tesseract-lang

# ===== 系统要求 =====
# - Python 3.8+
# - Windows 10/11 (推荐) / Linux / macOS
# - 内存: 8GB+ (推荐16GB+)
# - 显卡: NVIDIA GPU with CUDA支持 (可选，用于加速)
# - 硬盘: 至少5GB可用空间
